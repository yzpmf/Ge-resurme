## ✨ 项目简介

这是我的个人品牌展示网站，用于介绍我的背景、项目经历、技能专长和联系方式。网站整体采用 **天蓝色主调 + 奶油淡黄点缀**，中文字体使用楷体，营造清新、高级、有温度的视觉氛围。

- 🎯 **目标**：打造一张兼具专业度与个人风格的数字名片  
- 🧩 **内容**：关于我、项目经历、学习时间线、技能雷达、联系表单  
- 📱 **适配**：响应式设计，完美支持手机、平板、桌面端  

---

## 🎨 设计亮点

| 特性         | 说明 |
|--------------|------|
| **主色调**   | 深海天蓝 `#0A4A7B` → 浅空蓝 `#E6F2FF` 渐变体系 |
| **辅助色**   | 奶油淡黄 `#FFF3D4`、暖阳黄 `#FDE68A` 提亮关键元素 |
| **字体系统** | 中文：华文楷体 / 楷体；英文：Inter（无衬线混排） |
| **质感**     | 毛玻璃导航栏、柔和阴影、微光背景、大圆角卡片 |
| **动效**     | 滚动入场动画、技能条填充、打字机效果、悬浮抬升 |

---

## 🧱 技术栈

- **HTML5** – 语义化结构，SEO 友好  
- **CSS3** – 自定义变量、Flex/Grid 布局、渐变与毛玻璃效果  
- **原生 JavaScript (ES6)** – 交互逻辑（导航高亮、打字机、复制邮箱、表单提示）  
- **部署平台** – Netlify（自动 CI/CD + 自定义域名）  
- **版本管理** – Git & GitHub
